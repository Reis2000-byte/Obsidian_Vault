**id**: gives user information as well as all of the groups that the user is apart of

**whoami**: gives us our username

**pwd**: "print the working directory" gives us the current directory in which we are working

**mkdir**: makes another directory Example: "mkdir directory_name"

**ls**: lists the files inside of the local folder
	**Command Modifier**
		**-l**: long format
		**-a**: all

**"tab button"**: does auto completion

**cd**: to change into directories

**sudo**: "super user do" allows you to execute commands with admin privileges

**apt**: "advanced package tool" this is the command line interface for the package management system. Handles under the hood work of finding, downloading, and installing software from central repos.

**install**: specific **apt** command tells the manager to retrieve and set up the specified software

**vim** "file_name": opens vim creating that file if it does not exist

**mv**: used to move files. Can move files into other files

**cp**: used to copy files into other files

**touch**: used to create empty files or update time stamps

**cat**: used to read contents of files without opening them

**"|"**: is the way to take the output of one command and make it the input for another command

**echo**: is a command that prints something to the screen

**base64**: waiting for data to come in on standard input and wont stop running until it sees an end of file operator
	**Command Modifier**
		**-**: take all the input from standard input
		**-d**: decodes data from stdin

**curl**: tool used to do http requests to certain sites
